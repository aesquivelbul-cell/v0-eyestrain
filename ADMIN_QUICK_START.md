## 3-Step Quick Start

### 1️⃣ Login
- Go to `/login`
- Use admin credentials above
- Click "Sign In"

### 2️⃣ Go to Import Page
- After login, visit `/admin/import-data`
- Or click "Import Data" from homepage

### 3️⃣ Upload CSV & Import
- Click "Upload CSV"
- Select your survey file
- Wait for completion
- Done! Users created.

## Troubleshooting

**Can't access import page?**
→ Make sure you're logged in as admin

**Import not working?**
→ Check CSV format in guide

**Accounts not showing?**
→ Check browser console for errors
→ Clear cache and reload
